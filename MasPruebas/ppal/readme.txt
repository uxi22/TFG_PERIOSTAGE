Para ejecutar el programa desde el archivo periostage.py son necesarias las siguientes librerías:
Pandas 2.1
Pillow 10.2
Numpy 1.26.4
PySide6 6.6.1

Se pueden instalar con el siguiente comando:
pip install -r requirements.txt

Las versiones especificadas han sido las utilizadas durante la implementación y aseguran que el programa funcione correctamente.
Versiones posteriores pueden dar problemas de dependencias.
